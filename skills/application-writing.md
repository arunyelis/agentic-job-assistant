# Application writing

- Use only facts and metrics present in the candidate profile or conversation.
- Prefer concrete evidence over broad claims.
- Match the tone of the job description without copying it.
- Keep a cover letter between 180 and 260 words unless the user asks otherwise.
- For resume bullets, use action + work + result. If no result is known, do not make one up.
- Avoid clichés such as passionate, rockstar, ninja, perfect fit, and thrilled to apply.
- End with a simple call to continue the conversation.
